# Мобильный каталог музейных экспонатов (Python + HTML/CSS + XLSX)

## Цель
Веб‑приложение (мобильно‑дружелюбный интерфейс), которое читает данные из `.xlsx` и отображает каталог экспонатов: список, поиск/фильтры и карточку экспоната с фото.

## Входные данные
- `.xlsx` файл с листом, содержащим колонки:
  - `ID_Экспоната`
  - `Название_экспоната`
  - `Название_музея`
  - `Описание`
  - `Эпоха/Период`
  - `Категория`
  - `Дата_поступления`
  - `Фото_файл`
  - `История_связанная_с_Воронежем`
- Папка с фотографиями (имена файлов как в колонке `Фото_файл`).

## Выходные данные
- HTML‑страницы:
  - список экспонатов
  - карточка экспоната

## Назначение
Быстрый просмотр музейного фонда с телефона или ПК в браузере.

## Запуск (Windows)
1) Создай и активируй виртуальное окружение, установи зависимости:
```
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

2) Подготовь данные:
- положи Excel в `data/exhibits.xlsx`
- положи фото в `data/photos/` (например `1.jpg`)

3) Создай переменные окружения (пример в `example.env`).

4) Запусти:
```
python main.py
```

Открой в браузере: `http://127.0.0.1:8000`
